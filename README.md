 📝 Word Record Generator

A powerful Flask-based web application for generating professional Microsoft Word documents (.docx) with custom headings, text blocks, images, watermarks, page borders, and dynamic footers. Perfect for creating standardized records, reports, and templates with an intuitive visual preview system.

![Python](https://img.shields.io/badge/Python-3.7+-blue.svg)
![Flask](https://img.shields.io/badge/Flask-2.0+-green.svg)
![License](https://img.shields.io/badge/License-MIT-yellow.svg)

## ✨ Features

-  **Visual Document Builder** - Interactive web interface for constructing documents
-  **Image Support** - Insert and resize images with real-time preview
-  **Watermarking** - Add diagonal text watermarks (e.g., "CONFIDENTIAL")
-  **Page Borders** - Optional full-page borders with customizable spacing
-  **Smart Footers** - Dynamic footers that appear only on the last page
-  **Mobile Responsive** - Touch-friendly interface works on all devices
-  **Template Sharing** - Save and load document templates via unique 8-character codes
-  **Live Preview** - Multi-page visual preview before downloading
-  **Layout Options** - Normal (1") or Narrow (0.5") margins
-  **Rich Formatting** - Custom fonts, sizes, bold text, and alignment controls
